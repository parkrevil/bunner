# SPEC (Index)

> 본 문서는 L3 SPEC이며, 특정 기능에 대한 구현 계약(Implementation Contract)만을 정의한다.
> L1 불변식 및 L2 아키텍처 경계를 전제로 하며, 충돌 시 상위 문서가 우선한다.

본 문서는 Bunner의 L3 SPEC 문서군을 탐색하기 위한 인덱스다.

## SSOT 계층

- E0: AGENTS.md (집행)
- L1: FOUNDATION/INVARIANTS (존재 조건)
- L2: ARCHITECTURE (구조 경계)
- L3: SPEC (구현 계약)
